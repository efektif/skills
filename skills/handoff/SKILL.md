# Handoff

Use this skill to preserve continuity without dumping the whole chat.

## Include

- **Goal**: one sentence.
- **Current state**: what is done.
- **Changed files**: paths only.
- **Verification**: commands run and result.
- **Open items**: concrete next steps.
- **Cautions**: user changes, secrets, failing tests, environment blockers.

## Style

- Keep it under 80 lines.
- Prefer bullets.
- Include exact commands.
- Do not include secrets.
- Do not include irrelevant chat history.
